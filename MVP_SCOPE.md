# MVP Definition

## INCLUDED

### Authentication
- Email/password login
- JWT access token
- Refresh token
- Role-based access middleware

### Admin
- View talents
- View companies
- View jobs
- Update application stage
- Mark talent as job-ready

### Talent
- View courses
- Track progress %
- Edit profile
- Apply to jobs
- View application status

### Company
- Create job
- View applicants
- Move candidate in pipeline

### Matching Logic V1
Simple scoring formula:
40% skills match
20% assessment score
15% language match
10% availability
15% coach rating

## EXCLUDED (Post-MVP)
- Finance system
- AI CV parsing
- Integrations
- Advanced analytics
